---
title: placeholder的字体颜色大小
date: 2018-03-05 21:40:58
tags: [placeholder]
---

```css
input::-webkit-input-placeholder {
    /* WebKit browsers */
    font-size: 14px;
    color: #333;
}

input::-moz-placeholder {
    /* Mozilla Firefox 19+ */
    font-size: 14px;
    color: #333;
}

input:-ms-input-placeholder {
    /* Internet Explorer 10+ */
    font-size: 14px;
    color: #333;
}
```
